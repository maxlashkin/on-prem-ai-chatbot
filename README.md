PII Sanitizer (санитайзер персональных данных) — это программное обеспечение или инструмент, предназначенный для обнаружения и удаления (или маскировки) конфиденциальной информации из текстов и баз данных.
В 2025 году такие инструменты стали критически важными при работе с нейросетями (LLM), чтобы предотвратить утечку личных данных сотрудников или клиентов в облачные модели.
